Jekyll - мощный инструмент для создания статических сайтов.

**Общие теги с предыдущей статьей:** blog

Нажмите на тег "blog", чтобы увидеть все статьи с этим тегом.

## Полезные советы:

### 1. Используйте фронт-матери
```yaml
---
layout: post
title: "Мой пост"
date: 2024-01-15
tags: [tag1, tag2]
---
```

### 2. Организуйте структуру
```
_posts/       # Посты блога
_layouts/     # HTML шаблоны
_includes/    # Переиспользуемые компоненты
assets/       # CSS, JS, изображения
_data/        # YAML/JSON данные
```
