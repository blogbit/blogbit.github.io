Git - мощная система контроля версий.

**Общие теги с предыдущими статьями:** introduction, tutorial

## Основные команды Git:

### 1. Инициализация репозитория
```bash
git init
```

### 2. Добавление файлов
```bash
git add .
```

### 3. Коммит изменений
```bash
git commit -m "Описание изменений"
```
